# Materialkorb-Kartenmacher

Kleiner lokaler Prototyp, der aus einer Unterrichtsidee drei druckbare Karten macht: Material, Startsatz und Beobachtung.

## Zweck
- schneller Übergang von einer Idee zu einer greifbaren Arbeitskarte
- geeignet fuer GE, Lernwerkstatt und fachfremde Kolleg:innen
- bewusst ohne Konto, API oder Datenbank

## So oeffnen
1. Die Datei `index.html` direkt im Browser oeffnen.
2. Thema, Zielgruppe und Ziel anpassen.
3. Auf **Karten bauen** klicken.

## Was testen
- erscheinen drei Karten in der Vorschau?
- wird der Text unten passend aktualisiert?
- funktioniert **Text kopieren**?
- oeffnet **Drucken** die Druckansicht?

## Was waere ein Fehlschlag
- wenn die drei Karten nicht klar unterscheidbar sind
- wenn der Text zu generisch bleibt
- wenn die 60-Sekunden-Nutzung nicht ohne Erklaerung klappt
