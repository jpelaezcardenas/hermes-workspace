# Hermes Morning Cockpit Slice

## Zweck
Eine kleine lokale Morgenansicht für Hermes: Was ist heute relevant, welche Entscheidung steht an und was ist der nächste kleinste Schritt.

## Öffnen
`index.html` direkt im Browser öffnen.

## Test
1. Eine der drei Karten anklicken.
2. Eine Entscheidung wählen: `BEHALTEN`, `VERBESSERN` oder `WEGWERFEN`.
3. Prüfen, ob die Ansicht in unter 60 Sekunden verständlich bleibt.

## Woran es scheitert
- Wenn die Seite eher wie ein Dashboard als wie eine klare Morgenentscheidung wirkt.
- Wenn mehr als eine Entscheidung gleichzeitig offen bleibt.
- Wenn der Nutzen nicht in einem Blick erkennbar ist.

## Guardrails
- Keine Konten
- Keine externen APIs
- Keine personenbezogenen Daten
- Keine Installationen
- Kein Deploy
