# Vertretungsstunden-Assistent GE

Pfad: `/Users/zondrius/hermes-workspace/projects/night-lab/2026-06-10-vertretungsstunden-assistent-ge`

## Zweck
Ein lokaler 60-Sekunden-Prototyp fuer eine fachfremde oder spontane Vertretungsstunde in einer GE-Klasse. Er erzeugt keine personenbezogenen Daten, sondern eine einfache Ablaufkarte: Ankommen, Handlung, Hilfe, Fertig.

## Oeffnen
```bash
open /Users/zondrius/hermes-workspace/projects/night-lab/2026-06-10-vertretungsstunden-assistent-ge/index.html
```

## 60-Sekunden-Test
1. Seite oeffnen.
2. Situation waehlen: z. B. `unruhig`, `gar nichts sicher`, `5 Min`.
3. `Karte bauen` klicken.
4. Pruefen, ob die Karte in unter einer Minute eine vertretbare, nicht beschämende Mini-Stunde erklaert.
5. Optional: Drucken-Button testen.

## Was wuerde es scheitern lassen?
- Wenn echte Schuelernamen, Diagnosen oder Leistungswerte eingegeben werden muessen.
- Wenn die Karte zu viel Material voraussetzt.
- Wenn eine fachfremde Person die vier Schritte nicht sofort versteht.
- Wenn der Ablauf Druck, Bewertung oder Tempo erzeugt.
