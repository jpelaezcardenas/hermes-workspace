# Fuell-Leer-Spielraum GE

## Zweck
Ein isolierter lokaler Mini-Prototyp fuer den 60-Sekunden-Sichttest zur GE-Morgenidee `Mengen fuellen und leeren`.

Die App ist bewusst keine Lernstands-, Diagnose- oder Schuelerdaten-App. Sie zeigt nur eine einfache Handlung: Teile aus der Schale in den Becher fuellen, wieder leeren und fertig markieren.

## Oeffnen
```bash
open /Users/zondrius/hermes-workspace/projects/night-lab/2026-06-11-fuell-leer-spielraum/index.html
```

## 60-Sekunden-Test
1. Seite oeffnen.
2. `Fuellen` antippen: Ein Teil wandert in den Becher.
3. `Leeren` antippen: Ein Teil wandert zurueck in die Schale.
4. `Fertig` antippen: Abschluss- und Notizhinweis erscheint.
5. Pruefen: Versteht eine erwachsene Person ohne Erklaerung die Handlung und die Stop-Regel?

## Was wuerde den Prototyp scheitern lassen?
- Wenn er mehr erklaert als sichtbar macht.
- Wenn die Handlung am echten Tisch schneller ist als die App-Idee.
- Wenn echte Schuelerdaten, Diagnosen oder individuelle Fallinfos noetig waeren.
- Wenn daraus ohne Proof eine groessere Plattform oder V2 gebaut werden soll.

## Grenzen
Keine externen APIs, keine Accounts, kein Deploy, kein Commit, kein Push, keine sensiblen Daten.
