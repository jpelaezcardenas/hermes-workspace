# S-Kisten-Starter

Zweck: Ein lokaler Tiny-Prototyp fuer eine ruhige GE-Deutsch-/Wahrnehmungsstation zur S-Kiste. Er macht aus der Teacher-Morgenkarte eine visuelle 60-Sekunden-Handlung: Fokus waehlen, Karten sortieren, Beobachtungszettel drucken.

## Oeffnen
Oeffne `index.html` direkt im Browser.

## 60-Sekunden-Test
1. Seite oeffnen.
2. Einen Fokus waehlen: hoeren, fuehlen, zeigen oder sortieren.
3. Eine Gegenstandskarte antippen und pruefen, ob sie in `passt zu S` oder `passt nicht` landet.
4. Aufbau, Startsatz, Hilfen und Beobachtungszettel lesen.
5. Optional `Druckkarte oeffnen` testen.

## Was wuerde es scheitern lassen?
- Wenn die Seite laenger als eine Minute erklaert werden muss.
- Wenn sie wie ein reiner Textgenerator wirkt statt wie eine schnelle Stationskarte.
- Wenn echte Namen, Diagnosen oder personenbezogene Beobachtungen eingetragen werden.
- Wenn daraus eine grosse App, Datenbank oder externe Sammlung werden soll.

## Grenzen
Keine externen APIs, keine Konten, kein Deploy, kein Commit, kein Push, keine sensiblen Daten, kein Fuenferfeld.
