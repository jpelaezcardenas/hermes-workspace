# Lehrer-Morgenkarte GE

Kleine lokale Morgenkarte fuer GE ohne Daten, Login oder externe Dienste.

## Zweck
Dieses Mini-Prototype erzeugt drei knappe Aktivitaeten fuer den Unterrichtsmorgen. Es ist bewusst klein gehalten, damit man in unter einer Minute sieht, ob die Idee taugt.

## So oeffnen
- Datei `index.html` direkt im Browser oeffnen.
- Oder per Doppelklick aus dem Finder.

## Was testen
1. Bereich waehlen.
2. Dauer waehlen.
3. Niveau waehlen.
4. Auf **3 Mini-Aktivitaeten erzeugen** klicken.
5. Pruefen, ob die Vorschlaege knapp, praktisch und GE-tauglich wirken.

## Verifikation
- keine Namen
- keine Diagnosen
- keine externen APIs
- keine Installation noetig
- nur lokales HTML/CSS/JS
