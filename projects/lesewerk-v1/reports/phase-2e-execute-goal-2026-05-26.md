# LeseWerk Phase 2E - S-Tier Objektkern Execute Goal

## Oberziel

LeseWerk soll den naechsten sichtbaren Qualitaetssprung bekommen: aus der Wortschatzbibliothek wird ein weiterer konkreter, kindgerechter Mini-Reise-Ausbau. Nicht Masse, sondern S-Tier-Qualitaet.

## Fokus dieser Phase

`Tisch` wird als naechste volle Mini-Reise vorbereitet und umgesetzt, weil das Wort:

- stark alltags- und schulnah ist,
- im GE-Kontext sehr gut zeigbar ist,
- zur vorhandenen Objektwelt Sofa/Tasse/Teller passt,
- kurze Satzschablonen erlaubt,
- als naechster Baustein die spaetere Objektfamilie stabilisiert.

## Gewuenschtes Endprodukt

Eine nutzbare, lokal sichere Tisch-Mini-Reise:

1. Bild: "Schau den Tisch an."
2. Silbe/Wortaufbau: "Lies: Tisch."
3. Wort: "Lege oder waehle Tisch."
4. Satz: "Der Tisch ist da."
5. Mini-Geschichte: "Der Tisch ist da. Was passt?"

Die Reise soll im Lehrkraftbereich sauber auftauchen, profile-gated sein, ein lokales CSS-Symbol haben und keine externen Bilder verwenden.

## Harte Regeln

- Keine externe Bildquelle, keine METACOM-/ARASAAC-/Boardmaker-Platzhalter im Code.
- Keine Punkte, Timer, Diagnose, Ranking oder 1-10-Score im Kindmodus.
- `ch` in Tisch als didaktisches Risiko sichtbar machen, nicht unsichtbar vereinfachen.
- Keine grosse Refactor-Operation im selben Code-Slice.
- Tests erweitern, dann Build pruefen.
- Wenn Hermes blockiert: Stand dokumentieren, nicht endlos weiterlaufen.

## Task-Kette

### Phase 2E-A: Didaktischer Mini-Spec

Pruefe, ob `Tisch` im Vergleich zu `Brot`, `Keks`, `Heft` wirklich der beste naechste Code-Slice ist. Ergebnis: kurzer Spec mit Entscheidung, Gating, Satzidee, Distraktoren, Symbolidee, Risiken.

### Phase 2E-B: Code-Slice Tisch

Wenn 2E-A keine harte Bremse meldet: `Tisch` als Mini-Reise einbauen. Tests und Build muessen gruen sein.

### Phase 2E-C: Review und naechste 3 Slices

Pruefe das Ergebnis auf Kinderorientierung, technische Sauberkeit, didaktische Vorsicht und Design. Danach die naechsten drei Slices empfehlen.

## Erwarteter Qualitaetsmassstab

Das Ergebnis soll sich wie ein kleines eigenstaendiges Lese-Spiel anfuehlen: gross, klar, freundlich, bildhaft und ruhig. Lehrkraftlogik ist stark, aber im Kindmodus fast unsichtbar.
