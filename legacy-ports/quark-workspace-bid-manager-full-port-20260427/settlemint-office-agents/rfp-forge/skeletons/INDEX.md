# RFP Forge Skeleton Index

This directory contains buyer-side structural blueprints for procurement documents that the RFP Forge can generate.

## Setup Documents

| File | Purpose |
|---|---|
| `../setup/skeleton-definition.md` | Defines what an RFP Forge skeleton is and how to use it |
| `../setup/skeleton-scoring-rubric.md` | Scoring rubric used to review and improve skeleton quality |

## Current Structure

| Family | Variants | Markdown | DOCX | Purpose |
|---|---|---|---|---|
| RFP | 01-comprehensive, 02-focused, 03-compact | `RFP/markdown/` | `RFP/docx/` | Formal enterprise/private-sector request-for-proposal blueprints |
| RFI | 01-comprehensive, 02-focused, 03-compact | `RFI/markdown/` | `RFI/docx/` | Exploratory market-discovery request-for-information blueprints |
| Tender | 01-comprehensive, 02-focused, 03-compact | `Tender/markdown/` | `Tender/docx/` | Public-sector tender blueprints with rigid legal/evaluation structure |
| Questionnaire | 01-comprehensive, 02-focused, 03-compact | `Questionnaire/markdown/` | `Questionnaire/docx/` | Structured due-diligence and assessment questionnaire blueprints |
