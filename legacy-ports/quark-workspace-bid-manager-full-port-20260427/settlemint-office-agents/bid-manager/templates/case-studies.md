# Case Studies

> **Target length:** 3–4 pages (3–4 case studies, ~1 page each)
> **When to use:** RFP sections requesting references, prior experience, relevant projects, or proof of capability.
> **DALP sources:** Marketing materials, client success stories, public announcements
> **Important:** Only include case studies with client permission. Use anonymized versions if NDA prevents naming.

---

## How to Use This Template

1. Select 3–4 case studies most relevant to the client's industry, use case, and scale
2. Prioritize: same industry > same asset class > same geography > same scale
3. For each case study, complete all sections below
4. If client cannot be named, use format: "A Leading [Industry] Institution in [Region]"
5. Include quantified results wherever possible, percentages, time savings, cost reductions

## Selection Guide

When choosing case studies for a specific bid, prioritize by relevance:

| Priority | Criteria | Why |
|----------|----------|-----|
| 1 | Same industry as the prospect | Demonstrates domain expertise and regulatory understanding |
| 2 | Same asset class / use case | Shows direct capability match |
| 3 | Same geography / jurisdiction | Demonstrates knowledge of local regulatory environment |
| 4 | Similar scale / complexity | Provides confidence in handling comparable scope |
| 5 | Most impressive quantified results | Creates compelling proof points regardless of industry |
