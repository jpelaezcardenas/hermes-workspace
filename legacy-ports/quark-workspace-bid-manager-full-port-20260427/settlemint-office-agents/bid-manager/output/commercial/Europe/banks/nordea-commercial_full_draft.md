## Executive Summary

### Commercial Overview

SettleMint proposes a comprehensive DALP deployment to support Nordea's tokenized funds distribution programme across the Nordic region. The commercial framework is designed for Nordic banking requirements, with predictable annual licensing, transparent implementation pricing, and tiered support options.

### Recommended Commercial Model

| Component | Recommendation |
|-----------|----------------|
| Platform Tier | Enterprise |
| Deployment Model | Dedicated Cloud (AWS eu-north-1, Stockholm) |
| Support Tier | Premium |
| Implementation Scope | Full deployment with Nordic CSD integrations |
| Contract Term | 3-year initial |

### Investment Summary

- **Predictable annual licensing**: No per-transaction or per-token fees
- **Transparent implementation costs**: Fixed-price implementation
- **Premium support**: 12x7 coverage matching Nordic banking hours
