# Test Input 1: Digital Asset Custody RFI

## Scenario
A Tier 1 European bank has issued an RFI for a digital asset custody solution. They need to custody tokenized securities, CBDC, and crypto assets.

## Requirements from RFI:
1. Must support multiple asset classes (securities, CBDC, crypto)
2. Must integrate with their existing core banking system
3. Must support multi-party approval workflows
4. Must provide audit trails for regulatory compliance
5. Must support hardware security modules (HSM) for key management
6. Must have SOC 2 Type II certification
7. Must offer self-deployment option for data sovereignty
8. Should support webhook notifications for real-time events
9. Should provide SDK for custom integration
10. Must handle high transaction volumes (10k+ / day)

## Your Task
Respond to this RFI with:
- Executive summary (1 page)
- Compliance matrix for all 10 requirements
- 3 relevant client references
