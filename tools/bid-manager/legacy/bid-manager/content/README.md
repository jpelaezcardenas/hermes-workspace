## Numbering Convention

Content directories use a numbered prefix (`00-` through `08-`). Some numbers have multiple directories covering different aspects of the same domain area. The original `01-08` series contains the deep-dive technical content; the new `00-08` series provides topic-organized overlays for specific bid contexts.
