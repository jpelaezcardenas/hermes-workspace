# Bid Manager Enrichment Note: 2026-03-18

## Focus area
Lessons integration

## Change made
Added a new **Capability Boundary Rule** to `setup/writing-style.md`.

## What it says
Proposal writers must state explicitly whether a capability is native to DALP, handled through configuration, or dependent on external integration and project design choices.

## Why this matters
Recent lessons already emphasized that ambiguity around capability status weakens trust, even when the prose sounds polished. This update turns that lesson into an operational writing rule so future proposals are less likely to overclaim or blur delivery responsibility.
