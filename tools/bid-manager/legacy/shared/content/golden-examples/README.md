# Golden Examples

This directory stores reference-quality practice decks from the PPTX Builder training loop.

## Promotion Rule
Practice decks that score **35/40 or higher** in reviewer evaluation should be promoted here.

## Purpose
These examples serve as:
- quality benchmarks for future deck builds
- examples of strong slide selection and pacing
- references for copy density, visual balance, and narrative structure
- reusable standards when the builder needs a known-good pattern

## What to Store
For each promoted example, keep:
- the `.pptx` file
- an optional short note describing:
  - topic
  - score
  - strongest qualities
  - layouts used especially well

## Operating Principle
This is not a dumping ground. Only keep decks that are genuinely useful as standards worth imitating.
