# Lessons

- Generic Word output needs structure discipline, not proposal verbosity.
- Short documents should stay short.
- TOCs are useful for reports and reference docs, not for every memo.
- Word formatting defects are product defects for this agent.
- Most Word failures come from overbuilding the document, not underbuilding it.
- Strong first-page orientation does more work than ornamental formatting.
- Tables should carry facts, not smuggle in paragraphs.
- If a figure needs too much explanation, it is usually the wrong figure.
- Review the DOCX in Word every time. Markdown can hide layout defects.
