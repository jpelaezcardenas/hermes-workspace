# Common Defect Patterns

## Structural defects
- Title repeated again as the first section heading
- TOC included in a two-page memo
- Appendix carrying critical information that belongs in the body
- Headings that are vague or duplicative

## Writing defects
- Long warm-up paragraph before the point
- Consultant fog instead of a recommendation
- Repetition across summary and body with no added value
- AI markers, self-reference, or uncanny phrasing
- Em dashes sneaking back in

## Table defects
- More columns than the page can reasonably support
- Dense prose stuffed into cells
- No interpretation around the table
- Inconsistent phrasing row to row

## Figure defects
- Image dropped in without being introduced
- Caption missing even though the figure carries meaning
- Diagram too complex to read at normal zoom
- Screenshot includes irrelevant chrome and whitespace

## Conversion defects
- Blank page after cover or TOC
- TOC injected even when explicitly disabled
- Broken relative image path
- Oversized diagram pushing layout awkwardly
- Numbered headings or list behavior looking inconsistent

## Delivery defects
- DOCX reviewed only in markdown, not in Word
- Filename too vague for handoff
- Markdown source not saved with output
- Sensitive details left in when the audience did not need them
