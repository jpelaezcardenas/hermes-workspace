# Usage Examples

## Report
Use `templates/report-template.md`, then convert with `--toc`.

## Executive memo
Use `templates/executive-memo-template.md`, usually with `--no-toc`.

## Structured briefing
Use `templates/structured-briefing-template.md`.

## Q&A document
Use `templates/qa-template.md`.

## Meeting summary
Use `templates/meeting-summary-template.md`.

## Status update pattern
Use `templates/status-update-template.md` for recurring updates with:
- Current Status
- Changes Since Last Update
- Risks / Issues
- Decisions Needed
- Next Actions

## Research summary
Use `templates/research-summary-template.md` when synthesizing source material into a sendable document.

## Operating note / guidance memo
Use `templates/operating-note-template.md` for process guidance, rules, exceptions, and owner-based instructions.

## Formal external report
Use report structure plus `--mode full --toc --cover-json ...` only when a cover page materially improves the deliverable.
