# Icon Library

Two icon collections available for use in presentations:

## Lucide Icons (`lucide/`)
~111 SVG icons from the Lucide open-source library (MIT license).
See `lucide/README.md` and `lucide/INDEX.json` for the full list and categories.

**Best for:** Agenda icons, feature highlights, section markers, UI elements.

## SettleMint Custom Icons (`templates/assets/icons/`)
~30 domain-specific PNG icons for SettleMint/blockchain topics:
analytics, api, audit_trail, blockchain, certificate, clock, cloud, compliance,
contract, database, document, enterprise, exchange, globe, governance, integration,
lock, notification, partnership, platform, privacy, scale, security, settings,
smart_contracts, speed, timeline, tokenization, users, wallet

**Best for:** DALP-specific slides, blockchain concepts, product features.

## Note on Build Pipeline Usage

This is the raw SVG source library. For build pipeline usage, the canonical icon library is at `templates/icon-library/` which contains pre-rendered PNGs in brand colors.

## Usage
See `AGENTS.md → Icon Integration` for embedding instructions.
