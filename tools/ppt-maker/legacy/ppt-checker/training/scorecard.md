# PPTX Reviewer Scorecard Quick Reference

Use this as the fast calibration sheet when scoring decks against the 10-dimension rubric (/50 total).

## Dimensions

1. **Brand Compliance** — Checks template integrity, Figtree usage, colors, protected brand elements, and page numbering.
2. **Slide Selection / Layout Fit** — Checks whether the chosen layout matches the content and whether the deck uses the slide system intelligently.
3. **Text Quality** — Checks conciseness, clarity, hierarchy, active voice, and absence of filler or AI-sounding copy.
4. **Content Accuracy / Product Accuracy** — Checks DALP terminology, factual discipline, and whether claims stay inside verified capability.
5. **Visual Balance** — Checks whitespace, density, alignment, composition, and whether slides feel clean rather than cluttered.
6. **Narrative Flow / Narrative Clarity** — Checks whether the deck builds a coherent story in the right order with useful transitions.
7. **Audience Fit** — Checks whether the framing, depth, and terminology match the intended audience.
8. **Completeness** — Checks whether the deck is finished, populated, cleaned up, and free of leftovers or placeholders.
9. **Editability** — Checks whether the deck is built as a usable PPTX with native editable elements rather than flattened artifacts.
10. **Message Hierarchy** — Checks whether each slide lands one clear takeaway with the right headline and emphasis.

## Calibration Anchors

### 1 = Serious failure
The dimension is clearly broken and would materially hurt deck credibility, usability, or client readiness.

### 3 = Acceptable
The dimension is serviceable and usable, with visible issues but no major failure that blocks presentation.

### 5 = Excellent
The dimension is handled with unusual discipline and polish, with no meaningful cleanup needed.

## Verdict Thresholds

- **40–50** → Approve
- **30–39** → Revise
- **Below 30** → Rebuild
- **Either sub-score below 12/25** → Cannot Approve (regardless of total)

## Sub-Score Groupings

### Technical/Visual Sub-Score (/25)
Dimensions: Brand Compliance (1) + Slide Selection (2) + Visual Balance (5) + Completeness (8) + Editability (9)

Covers: font correctness, color compliance, logo integrity, element spacing, text overflow, element overlap, title placement, header/footer, image quality, density, alignment, template discipline, cross-slide consistency, color contrast.

### Content/Messaging Sub-Score (/25)
Dimensions: Text Quality (3) + Content Accuracy (4) + Narrative Flow (6) + Audience Fit (7) + Message Hierarchy (10)

Covers: copy quality, claim accuracy, DALP terminology, story structure, audience targeting, tone, style, positioning alignment.

## Quick Scoring Reminder

- Start conservative: a **3** is normal.
- Use **4** only when the deck is clearly above baseline.
- Use **5** rarely.
- If you are torn between two scores, use the lower one unless the higher score is clearly earned.
- Hard failures should be reserved for issues that truly block client use.
- Always report both sub-scores alongside the total in the Score Card.
- A deck with a 20/25 visual score but 10/25 content score should NOT be approved — the messaging is broken.
- A deck with a 10/25 visual score but 20/25 content score should NOT be approved — the production is broken.
